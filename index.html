Welcome to KodNest Premium Build System

This is a complete design system built for B2C product companies that value calm, intentional design over flashy aesthetics. Every element has been carefully crafted to work together as one coherent system.

Design Philosophy

Calm, intentional, coherent, confident. No gradients, no glassmorphism, no neon colors, no animation noise.

Color System

Maximum 4 colors across the entire system. Off-white background, deep red accent, muted supporting colors.

Typography

Serif headings with generous spacing. Clean sans-serif body at 17px with 1.7 line-height for optimal readability.

Spacing Scale

Consistent scale: 8px, 16px, 24px, 40px, 64px. Never use random spacing. Whitespace is part of design.

Global Layout

Every page follows: Top Bar → Context Header → Workspace + Panel → Proof Footer. No exceptions.

Components

Predictable, consistent components. Same hover effects, border radius, and transitions everywhere.

File Structure

/styles ├── tokens.css # Design tokens (colors, spacing, typography) ├── base.css # Base styles and typography ├── layout.css # Global layout structure ├── components.css # Component styles └── main.css # Entry point (imports all) /index.html # This page /demo.html # Full demo page with layout structure /components.html # Component reference /README.md # Complete documentation

Getting Started

Import the design system in your HTML:

<link rel="stylesheet" href="styles/main.css">
